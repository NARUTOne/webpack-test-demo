## PROJECT NAME
> 本项目只做webpack + react + antd 学习使用测试 :fire: :bug:

- webpack: 3.8.1
- react: ^15.6.2
- react-router: ^3.0.2 
- antd : 2.13.7

独立的前端代码包，接通数据源后（`src/utils/config.js`）即可本地运行，放置服务器后端环境下即可部署

## 安装

```sh
$ git clone </path/to/repository>

$ cd repository

$ npm install
```

## 本地开发环境运行

```sh
$ npm start
```

## 打包发布

```sh
$ npm run build
```
